## 📖 Overview

## 📚 Subject Descriptions

### 🔧 CCL — Competitive Coding Lab
Focused on **interview preparation and DSA (Data Structures & Algorithms)** using C++. Contains solutions to competitive programming problems and LeetCode-style challenges covering topics like arrays, recursion, dynamic programming, graphs, and more.

### 💻 COA — Computer Organization & Architecture
Covers low-level programming experiments in C++ that explore CPU organization, instruction sets, memory hierarchies, and fundamental architectural concepts.

### 📊 DAV — Data Analysis & Visualization
A collection of Jupyter Notebooks built with Python, covering data preprocessing, exploratory analysis, statistical methods, and visual storytelling using libraries like Pandas, Matplotlib, and Seaborn. Includes associated `data/` and `output/` directories.

### 🌐 WP — Web Programming
Weekly lab exercises focused on frontend and full-stack web development. Organized by week, these cover progressively advanced topics in HTML, CSS, and JavaScript.

## 🚀 How to Use

**Clone the repository:**
```bash
git clone https://github.com/<your-username>/sem4-exp.git
cd sem4-exp
```

**Navigate to a subject:**
```bash
cd dav          # for Data Analysis & Visualization
cd wp/week3.1   # for a specific Web Programming week
```

**Run a Jupyter Notebook (DAV):**
```bash
jupyter notebook dav/exp1.ipynb
```

**Compile a C++ file (CCL / COA):**
```bash
g++ ccl/exp4_3.cpp -o exp4_3 && ./exp4_3
```
